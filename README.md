# ⚡ SWAY.IN Machine Learning Trading Terminal

The stock market is highly volatile and influenced by multiple dynamic factors, making it difficult for traders and investors to predict price movements accurately. Traditional methods rely heavily on manual analysis and experience, which may not always yield consistent results.

This project aims to develop an AI-based system that utilizes real-time market data and machine learning techniques to analyze patterns and predict future price movements of major indices like **NIFTY 50** and **SENSEX**, as well as major Indian equities. The system also provides a comprehensive visual dashboard and automated trading signals to assist users in their decision-making process.

## 🎯 Objectives
- **Collect real-time stock market data** using external data APIs such as Alpha Vantage and `yfinance`.
- **Preprocess and normalize** financial time-series data sequences.
- **Implement technical indicators** like EMA, RSI, MACD, and Bollinger Bands for algorithmic trend analysis.
- **Build an LSTM-based deep learning model** optimized for continuous neural price prediction.
- **Develop an interactive dashboard** using Streamlit with real-time UI/UX features.
- **Generate BUY/SELL/HOLD signals** explicitly mapped onto interactive Candlestick charts based on prediction and momentum indicators.
- **Ensure system reliability & fallback** using a hybrid, zero-downtime API connection approach that seamlessly switches data sources on limits.

---

## 🚀 Key Features Built into SWAY.IN

* **Timezone-Aware Live Reloading:** Automatically detects if the Indian/US market is actively ticking or closed, dynamically pausing API calls to prevent dead looping.
* **Premium Trading Terminal UI:** Multi-tab layout featuring a main interactive SuperChart, Neural Network analytics, Fundamental Company Profiles, and a Live Global News feed!
* **Trade Signal Overlays:** Uses algorithm logic (`RSI<30`, `RSI>70`) to strictly instruct when to enter (**Buy**) and when to exit (**Sell**) by painting signals directly onto price wicks.
* **LSTM Forecasting Engine:** Uses `tensorflow.keras` strictly trained over the last 60 temporal periods to generate a deterministic $ Target Price for the next incoming candle.
* **AI Backtester Sandbox:** Dedicated isolated environment that simulates a ₹100,000 algorithmic portfolio across historical intervals to measure hypothetical Profit/Loss% generated by the AI strategy.

## 🖥️ Installation and Usage

1. **Clone the Repository**
   ```bash
   git clone https://github.com/SwayamNerkar/SWAYIN.AI-index-price-accuracy.git
   cd SWAYIN.AI-index-price-accuracy
   ```

2. **Activate your Virtual Environment**
   ```bash
   # Windows
   .\venv\Scripts\activate
   # Mac/Linux
   source venv/bin/activate
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Launch the SWAY.IN Terminal!**
   ```bash
   streamlit run app.py
   ```
